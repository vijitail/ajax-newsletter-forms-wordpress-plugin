An Ajax newsletter forms plugin that works well with the Newsletter Plugin and can also submit to external URL.

== Description ==
Ajax Newsletter Forms is useful if you don\'t want to redirect your user to a thank you page or refresh the page. It is fully compatible with the Newsletter Plugin and can also submit forms to other URL. 

If you want to build custom forms you can use the name values in your inputs as follows:
- first name: anf_name
- last name: anf_lname
- email: anf_email
- list number : anf_list_num

== Installation ==
1. Upload \"test-plugin.php\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Place \"do_action( \'plugin_name_hook\' );\" in your templates.
